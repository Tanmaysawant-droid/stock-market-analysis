# 📊 Stock Market Analysis using Python

This project is a simple attempt to understand stock market behavior using Python.  
I used libraries like NumPy, Pandas, Matplotlib, and SciPy to analyze historical stock data and visualize trends.

The goal of this project was to explore how stock prices move, measure risk, and identify patterns using basic data analysis techniques.

---

## 🚀 What this project does

In this project, I:

- Collected real stock data using `yfinance`
- Cleaned and processed the data using `pandas`
- Calculated important metrics like daily returns and volatility
- Used moving averages to understand trends
- Applied linear regression to analyze the overall direction of the stock
- Visualized everything using `matplotlib`

---

## 🛠️ Tools & Libraries

- **Pandas** → for handling and analyzing data  
- **NumPy** → for numerical calculations  
- **Matplotlib** → for plotting graphs  
- **SciPy** → for regression and statistical analysis  
- **yfinance** → for fetching stock data  

---

## 📊 Key Features

- 📌 **Moving Averages (20 & 50 days)**  
  Helps in identifying short-term and long-term trends  

- 📌 **Daily Returns**  
  Shows how much the stock price changes each day  

- 📌 **Volatility Analysis**  
  Measures how risky or stable the stock is  

- 📌 **Trend Analysis (Regression)**  
  Gives a general idea of whether the stock is moving upward or downward  

- 📌 **Visualizations**  
  Includes multiple graphs to better understand the data
